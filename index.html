Get Started in Seconds

Copy any agent config and integrate it into your project with just a few lines of code.

# Install AgentMD CLI
npm install -g agentmd-cli

# Search and download agent config
agentmd search "code assistant"
agentmd get gpt4-code-assistant

# Use in your project
import { Agent } from './agent.md'
const assistant = new Agent(config)
